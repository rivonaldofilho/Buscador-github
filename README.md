# GitHub Profile Finder


## Descrição do Projeto
O **GitHub Profile Finder** é uma aplicação web simples que permite buscar informações de perfis públicos do GitHub utilizando a API oficial do GitHub. O usuário pode inserir um nome de usuário do GitHub e visualizar informações como:
- Foto de perfil
- Nome
- Biografia
- Número de repositórios públicos

O projeto foi desenvolvido com o objetivo de praticar habilidades de desenvolvimento web, incluindo consumo de APIs e manipulação do DOM.

---

## Tecnologias Utilizadas
- **HTML5**: Estrutura da aplicação.
- **CSS3**: Estilização da interface, com foco em um design limpo e responsivo.
- **JavaScript**: Lógica para consumir a API do GitHub e exibir os dados do perfil.

---

## Funcionalidades
- Campo de entrada para o nome de usuário do GitHub.
- Botão para buscar informações do perfil.
- Exibição de informações do perfil em um card estilizado.
- Tratamento de erros para casos de usuários não encontrados.

---

## Habilidades Praticadas
- Consumo de APIs REST (GitHub API).
- Manipulação do DOM com JavaScript.
- Estilização com CSS para criar uma interface agradável e responsiva.
- Tratamento de erros e validação de entradas do usuário.
